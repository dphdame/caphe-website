**Size:** 128x128 pixels (will be displayed at 48-64px, so keep simple)

## Style Notes

- Keep icons simple and recognizable at small sizes
- Use consistent stroke width across all icons
- Avoid fine details that won't scale well
- Consider creating a cohesive set that looks unified when displayed together
- Icons should feel "institutional" and "professional" rather than playful
- Reference style: Similar to Font Awesome or Heroicons professional sets
