# STYLE SPECIFICATION: CAPHE County ROI Reports

**Created**: January 19, 2026
**Based on**: 6 content samples (~4,500 words total)
**Sources**: Homepage, About, Methods Lab, LHA Calculator, Tools pages

---

## TARGET AUDIENCE PROFILE

### Primary Recipients
- **Local Health Authority (LHA) Directors** - Need data for budget justification
- **County Health Officers** - Want evidence for program advocacy
- **Public Health Program Managers** - Seek presentation-ready materials
- **Budget Analysts** - Require clear cost-benefit documentation

### Audience Characteristics
- Public health domain expertise, but **limited economics training**
- Time-constrained; need scannable, actionable information
- Seeking credible evidence for internal/external stakeholders
- Value practical application over theoretical depth

---

## 1. VOICE ATTRIBUTES

### Formality Level
**Professional-accessible blend** - Technical credibility without jargon walls.

The voice sits between academic rigor and practitioner accessibility. It uses precise terminology when necessary but immediately explains or contextualizes technical terms.

**Example**: "Based on Lewbel IV estimation on California county data (2003-2023): each $10 per capita spending reduces mortality by 9.16 deaths per 100,000."

### Authority Stance
**Credible guide, not lecturer** - Establishes expertise through:
- Specific methodology references (Lewbel IV, panel data)
- Peer-reviewed citations with year and publication
- Concrete numbers (9.16 deaths per 100,000, not "significant reduction")

Does NOT lecture or oversell. Acknowledges limitations explicitly.

**Example**: "Results are estimates based on historical county-level data. Actual ROI may vary based on intervention type and implementation."

### Reader Relationship
**Collaborative peer** - Positions CAPHE as a resource partner, not authority figure.

Uses "your county," "for presentations," "use these estimates" language that assumes the reader is the expert in their context—CAPHE provides the tools.

**Example**: "Upload spending data to receive a customized PDF report showing the return on investment for public health programs."

## 2. SENTENCE PATTERNS

### Metrics
- Average sentence length: **18-22 words**
- Range: **8-40 words**
- Simple/Compound/Complex: **50%/30%/20%**
- Fragment usage: **Rare** (professional documents avoid fragments)

### Rhythm Description
Opens with clear, direct statements. Follows with supporting detail. Closes sections with actionable takeaway or practical note.

**Pattern**: Claim → Evidence → Application

### Punctuation Style
- Em dashes: **Light** (use parentheses instead for asides)
- Semicolons: **Rare** (prefer shorter sentences)
- Parentheticals: **Moderate** (for technical clarifications)
- Oxford comma: **Yes**
- Colons: **Frequent** (for lists and explanations)

**Example rhythm**: "The model uses Lewbel IV estimation on California county data (2003-2023). Each $10 per capita spending reduces mortality by 9.16 deaths per 100,000. Social value uses the HHS Value of Statistical Life ($13.6M)."

## 3. VOCABULARY GUIDELINES

### Preferred Terms
Use these consistently in ROI reports:

| CAPHE Term | NOT |
|------------|-----|
| Lives saved | Deaths averted |
| Benefit-cost ratio | Return on investment ratio |
| Per capita spending | Per person expenditure |
| Mortality reduction | Death rate decrease |
| Social value | Societal benefit |
| Value of Statistical Life (VSL) | Statistical value of life |
| Peer-reviewed research | Academic studies |
| Causal inference | Correlation analysis |
| Evidence-based | Science-backed |
| Data-driven | Metrics-focused |
| Estimates | Predictions |
| California county data | Statewide information |

### Words to Avoid
Never use these (they undermine credibility or overstate):

- "Proven" (use "estimated," "research shows")
- "Guaranteed" (use "based on historical data")
- "Revolutionary" or "groundbreaking"
- "Best-in-class" or superlatives
- "Just" (minimizing)
- "Simple" or "easy" (when describing methodology)
- "Utilize" (use "use")
- Undefined acronyms on first use

### Specificity Level
**Concrete over abstract** - Always include:
- Specific coefficients (9.16, not "approximately 9")
- Dollar amounts ($13.6M, not "millions")
- Time periods (2003-2023, not "recent years")
- Geographic scope (California county data, not "local data")

### Intensifier Usage
**Low** - Let the data speak. Avoid:
- Very, really, extremely
- Significant (unless statistically defined)
- Substantial (be specific instead)

**Exception**: "Key finding" is acceptable to highlight primary results.

### Contraction Style
**Avoid in formal reports** - Use contractions only in informal contexts (tooltips, help text). Reports should maintain full forms:
- "does not" not "doesn't"
- "is not" not "isn't"

## 4. STRUCTURAL PREFERENCES

### Report Section Order
1. **Executive Summary** - Key metrics at a glance
2. **Key Findings** - 3-4 headline numbers with context
3. **Spending Breakdown** - Data visualization with table
4. **Methodology** - Source, coefficient, limitations
5. **Citation** - Full academic reference
6. **Contact** - CAPHE website and email

### Paragraph Length
- Average: **2-3 sentences** (40-60 words)
- Range: **1-4 sentences**
- Never exceed 5 sentences per paragraph

### Transition Style
**Implicit flow with section headers** - Minimize explicit transitions ("Furthermore," "Additionally"). Let clear headers organize the flow.

When transitions are needed:
- "Based on this analysis..."
- "This estimate reflects..."
- "For context..."

### Data Presentation
- **Round appropriately**: Lives saved to 1 decimal (194.2), BCR to whole number (125:1)
- **Use consistent formats**: $X.XM, $XXK, XX%
- **Include units**: "per year," "per 100,000 population"
- **Provide comparison**: "vs. $13.6M federal VSL"

---

## 5. RHETORICAL MOVES

### How to Open Sections
Lead with the most important information (inverted pyramid):

**DO**: "Sacramento County's public health investment of $21.2M is estimated to save 194 lives annually."

**DON'T**: "This report presents an analysis of public health spending..."

### How to Handle Evidence
- Integrate citations naturally: "Based on Cholette, Patton & Zaragoza-Gonzalez (2026)..."
- Provide specific source: Working paper, journal, year
- Explain methodology briefly: "using Lewbel IV estimation"
- Include time period: "(2003-2023)"

**Pattern**: "[Finding] based on [Author (Year)] using [method] on [data source]."

### How to Present Limitations
Be transparent but not undermining:

**DO**: "These estimates are based on historical county-level data. Actual results may vary based on program implementation."

**DON'T**: "These numbers might be completely wrong in your situation."

### How to Conclude
End with practical application or next step:

**DO**: "Full methodology available in the attached working paper."

**DON'T**: "In conclusion, we have shown that..."

### Prohibited Patterns
- Marketing language ("best," "leading," "premier")
- Vague claims without numbers
- Passive voice when actor is known
- Run-on sentences
- Undefined jargon

## 6. ANNOTATED SAMPLES

### Sample 1: Methodology Explanation (from LHA Calculator)

**PASSAGE:**
> "Using California county data (2003-2023): each $10 per capita spending is estimated to reduce mortality by 9.16 deaths per 100,000. Source: Cholette, Patton & Zaragoza-Gonzalez (2026), SSRN Working Paper"

**ANNOTATIONS:**
- Opens with data source and time period (specificity)
- Uses precise coefficient (9.16, not "about 9")
- Includes unit of measurement (per 100,000)
- Full citation with year and publication type
- Passive voice acceptable here (focus is on finding, not researcher)

### Sample 2: Limitations Acknowledgment (from Tools page)

**PASSAGE:**
> "The coefficient represents an average effect across all public health activities. Specific interventions may have higher or lower returns."

**ANNOTATIONS:**
- Direct acknowledgment of limitation
- Explains why limitation exists (average effect)
- Provides balanced framing (higher OR lower)
- Does not undermine credibility
- Two short sentences for clarity

## QUICK REFERENCE CARD

When writing CAPHE ROI reports, remember:

**DO:**
- Lead with key numbers (lives saved, BCR)
- Include specific methodology references with years
- Provide context for statistics ("vs. federal VSL")
- Use consistent formatting for currency and percentages
- Acknowledge limitations transparently
- End sections with practical implications

**DON'T:**
- Use marketing language or superlatives
- Leave acronyms undefined
- Present estimates as guarantees
- Use vague quantifiers ("many," "significant")
- Bury key findings in paragraphs
- Overload with technical jargon

**SIGNATURE PHRASES:**
- "Based on [Author (Year)] using [method]..."
- "Each $[X] per capita spending reduces mortality by..."
- "Social value calculated using the HHS Value of Statistical Life ($13.6M)"
- "These estimates are for [use case]. Formal evaluations require..."

## IMPLEMENTATION NOTES

**Best for**:
- Executive summaries and key findings sections
- Data presentation with visualizations
- Methodology explanations for non-economists

**Challenging for**:
- Highly technical appendices (may need more formal academic tone)
- Marketing materials (this voice is too restrained)

**Context adjustments**:
- **Presentations**: Can be slightly more informal, shorter sentences
- **Technical appendix**: Can include more statistical detail
- **Email summary**: Can use contractions, more conversational
