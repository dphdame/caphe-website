# CAPHE Website - Project Brief

## Overview
CAPHE (California Association of Public Health Economists) is a professional organization website providing resources, programs, and educational content for public health economists in California.

## Core Requirements

### Membership Tiers
1. **Public** - No account required
   - Access to foundational content
   - Public labs in Methods Lab

2. **Community (Affiliate)** - Free account required
   - Access to community-tier content
   - 16 advanced methodology labs
   - California case studies
   - Webinar recordings

3. **Professional (Member)** - Paid/approved membership
   - Full access to all content
   - Professional-only labs
   - Working groups
   - Peer review sessions

### Key Features
- **Methods Lab**: Interactive labs teaching causal inference and economic evaluation
- **Membership System**: Tiered access control with Supabase authentication
- **Programs**: Webinars, workshops, working groups, peer review sessions
- **Resources**: External resources and curated content

## Technical Stack
- **Frontend**: Static HTML/CSS/JS served from Express
- **Backend**: Node.js/Express server
- **Authentication**: Supabase Auth
- **Email**: Brevo (formerly Sendinblue)
- **Hosting**: Heroku
- **Domain**: caphegroup.org

## Project Goals
1. Provide accessible health economics education
2. Build a community of practice for California public health economists
3. Offer tiered content to encourage membership
4. Maintain professional, academic-quality presentation
