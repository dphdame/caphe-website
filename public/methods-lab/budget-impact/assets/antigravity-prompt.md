# Antigravity Prompt: Budget Impact vs Cost-Effectiveness DAG

## Prompt

Clean, professional comparison diagram on white background showing two complementary analysis frameworks side by side.

Layout: Two distinct sections, left and right, connected by a central "INTERVENTION" node at top.

**Left Section - Cost-Effectiveness Analysis (CEA):**
- Top: Rounded rectangle node "INTERVENTION" in teal (#0D9488), white text
- Arrow pointing down to "HEALTH GAINS (QALYs)" node in green (#2E7D32), white text
- Arrow pointing down to "COST" node in gray (#424242), white text
- Both arrows converge at bottom to "VALUE RATIO ($/QALY)" node in primary blue (#0041A5), white text
- Label above section: "Cost-Effectiveness Analysis" in small caps

**Right Section - Budget Impact Analysis (BIA):**
- Top: Same "INTERVENTION" node connects to this section
- Arrow pointing down to "TOTAL SPENDING" node in gold (#F9A825), dark text
- Arrow pointing down to "BUDGET CONSTRAINT" node in orange (#F57C00), white text
- Both converge at bottom to "AFFORDABILITY" node in primary blue (#0041A5), white text
- Label above section: "Budget Impact Analysis" in small caps

**Center connecting element:**
- A horizontal dashed line connecting the two bottom result nodes
- Small text below: "Both needed for complete decisions"

Arrows:
- Solid gray (#424242) arrows for the main flow
- Small arrowheads, clean lines
- 2px stroke weight

Style: Flat design, no gradients, no shadows. Minimalist infographic style.
Montserrat font, semi-bold weight. Arrows have small arrowheads.
Nodes have 8px rounded corners and generous padding.

Key visual message: CEA and BIA are parallel analyses that each answer different questions, but both feed into the final decision.

## Notes

This diagram illustrates the complementary nature of cost-effectiveness analysis (asking "Is this worth the money?") and budget impact analysis (asking "Can we afford this?"). The parallel structure emphasizes that these are independent questions that must both be answered for informed decision-making.

The color coding helps distinguish:
- Teal for value-based concepts (CEA perspective)
- Gold/orange for financial constraint concepts (BIA perspective)
- Primary blue for final decision points in both frameworks
