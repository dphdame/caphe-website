# Antigravity Prompt: Comparator Choice DAG

## Prompt

Clean, professional directed acyclic graph (DAG) diagram on white background.
4 rounded rectangle nodes arranged in a diamond formation:

**Top center:**
- "BUDGET" node in Primary Blue (#0041A5) with white text

**Middle row (left and right):**
- "Program A" node in Teal (#0D9488) with white text (left side)
- "Program B" node in Teal (#0D9488) with white text (right side)

**Bottom center:**
- "Health Outcomes" node in Gold (#F9A825) with dark text (#1C1C1C)

Arrows:
1. From BUDGET to Program A (solid gray #424242, indicating constraint)
2. From BUDGET to Program B (solid gray #424242, indicating constraint)
3. From Program A to Health Outcomes (solid gray #424242)
4. From Program B to Health Outcomes (dashed gray #424242, labeled "Alternative")

Add a "VS" badge between Program A and Program B in a red circle (#E53935) with white text.

Style: Flat design, no gradients, no shadows. Minimalist infographic style.
Montserrat font, semi-bold weight. Arrows have small arrowheads.
Nodes have 8px rounded corners and generous padding.

The key visual message: choosing one program means NOT choosing another. The budget constraint forces a tradeoff.
