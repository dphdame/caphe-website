# Antigravity Prompt: History & Maturation Threats DAG

## Prompt

Clean, professional directed acyclic graph (DAG) diagram on white background.
5 rounded rectangle nodes showing two confounding pathways:

**Top row (two confounders):**
- "EXTERNAL EVENTS (History)" node in Primary Blue (#0041A5) with white text (left)
- "TIME / AGING (Maturation)" node in Primary Blue (#0041A5) with white text (right)

**Middle:**
- "Program" node in Teal (#0D9488) with white text (center)

**Bottom:**
- "Health Outcomes" node in Gold (#F9A825) with dark text (center)

**Arrows:**
- Solid gray arrow from EXTERNAL EVENTS down to "Program"
- Solid gray arrow from EXTERNAL EVENTS down to "Health Outcomes"
- Solid gray arrow from TIME/AGING down to "Program"
- Solid gray arrow from TIME/AGING down to "Health Outcomes"
- Dashed green arrow with "?" from "Program" to "Health Outcomes"

Style: Flat design, no gradients, no shadows. Minimalist infographic style.
Montserrat font, semi-bold weight. Arrows have small arrowheads.
Nodes have 8px rounded corners and generous padding.

Key visual: Two separate threats (History on left, Maturation on right) both create spurious associations. The "?" on the causal arrow shows uncertainty about whether the program actually worked.
