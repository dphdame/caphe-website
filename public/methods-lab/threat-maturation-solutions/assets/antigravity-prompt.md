# Antigravity Prompt: ITS Design Diagram

## Prompt

Clean, professional diagram showing Interrupted Time Series (ITS) design concept on white background.

Create a conceptual diagram with these elements:

**Left side:** A time series line from 2014-2024 with:
- Pre-intervention period (2014-2018): Teal (`#0D9488`) solid line trending upward
- Intervention marker at 2019: Blue (`#0041A5`) vertical dashed line labeled "Program Launch"
- Post-intervention period (2019-2024): Gold (`#F9A825`) solid line trending upward (steeper than pre)
- Counterfactual line: Teal dashed line continuing pre-intervention slope

**Right side:** Two boxes stacked:
- Top box (teal border): "Pre-Program Slope = Maturation"
- Bottom box (gold border): "Post-Program Slope = Maturation + Program Effect"

**Arrow annotation:** Between actual post-line and counterfactual, label reading "Program Effect = Slope Change"

Style: Flat design, no gradients, minimal shadows. Academic infographic style.
Montserrat font, semi-bold weight. Clean axis lines with tick marks.
White background with subtle gray gridlines.
