# Antigravity Prompt: Maturation Threat DAG

## Prompt

Clean, professional directed acyclic graph (DAG) diagram on white background.
4 rounded rectangle nodes arranged in a horizontal flow with one node above:

**Top center (confounder):**
- "TIME / NATURAL PROGRESSION" node in Primary Blue (#0041A5) with white text

**Bottom row (left to right):**
- "Program Launch" node in Teal (#0D9488) with white text
- "Health Outcomes" node in Gold (#F9A825) with dark text

**Arrows:**
- Solid gray arrow from TIME down to "Program Launch" (programs often launched when things are bad)
- Solid gray arrow from TIME down to "Health Outcomes" (outcomes naturally improve over time)
- Dashed green arrow with "?" from "Program Launch" to "Health Outcomes" (the causal effect we want to measure)

Style: Flat design, no gradients, no shadows. Minimalist infographic style.
Montserrat font, semi-bold weight. Arrows have small arrowheads.
Nodes have 8px rounded corners and generous padding.

Key visual: The TIME node looms above, showing it drives both when programs launch AND natural improvement in outcomes. The dashed arrow with "?" emphasizes we can't tell if the program worked or if improvement was inevitable.
